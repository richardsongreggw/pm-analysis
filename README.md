# PM Analysis Workspace

Product management analysis workspace for business cases, market research, and data-driven decision making.

## Structure

- `notebooks/` - Jupyter notebooks for analysis work
- `data/` - Data files (gitignored for privacy)
  - `raw/` - Original datasets
  - `processed/` - Cleaned/transformed data
- `scripts/` - Python scripts for data processing
- `reports/` - Exported analyses (PDF/HTML)
- `templates/` - Reusable notebook templates

## Setup

1. Install dependencies:
   ```bash
   pip3 install -r requirements.txt
   ```

2. Start Jupyter:
   ```bash
   jupyter notebook
   ```

## Usage

Create new analyses in the `notebooks/` directory. Use templates from `templates/` as starting points.
